# 🧪 Proyecto de Pruebas Unitarias - Módulo 14

Este proyecto consiste en la implementación de una suite de pruebas unitarias utilizando **JUnit 5** para validar la lógica de registro de usuarios en un sistema educativo.

## 🚀 Objetivo
Asegurar que la clase `ValidadorRegistro` cumple con las reglas de negocio establecidas, bloqueando datos inválidos y permitiendo el registro con datos correctos.

## 📋 Casos de Prueba Implementados
Se han desarrollado los siguientes 5 retos de testing:

1.  **Nombre Vacío:** Verifica que el sistema rechaza nombres sin contenido.
2.  **Password Justa (8 caracteres):** Valida que el límite mínimo de 8 caracteres sea aceptado.
3.  **Password Corta:** Asegura que contraseñas menores a 8 caracteres sean bloqueadas.
4.  **Email sin Arroba:** Comprueba que el formato de correo sea validado correctamente.
5.  **Edad Límite (16 años):** Verifica que la edad mínima permitida (16) sea aceptada por el sistema.
